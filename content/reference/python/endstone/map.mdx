## endstone.map [#endstone.map]

**Classes:**

- [**MapCanvas**](#endstone.map.MapCanvas) – Represents a canvas for drawing to a map. Each canvas is associated with a specific MapRenderer and represents that renderer's layer on the map.
- [**MapRenderer**](#endstone.map.MapRenderer) – Represents a renderer for a map.
- [**MapView**](#endstone.map.MapView) – Represents a map item.

### MapCanvas [#endstone.map.MapCanvas]

Represents a canvas for drawing to a map. Each canvas is associated with a specific MapRenderer and represents that renderer's layer on the map.

**Methods:**

- [**draw_image**](#endstone.map.MapCanvas.draw_image) – Draw an image to the map. The image will be clipped if necessary.
- [**get_base_pixel_color**](#endstone.map.MapCanvas.get_base_pixel_color) – Get a pixel from the layers below this canvas.
- [**get_pixel_color**](#endstone.map.MapCanvas.get_pixel_color) – Get a pixel from the canvas.
- [**set_pixel_color**](#endstone.map.MapCanvas.set_pixel_color) – Draw a pixel to the canvas.

**Attributes:**

- [**map_view**](#endstone.map.MapCanvas.map_view) (<code>[MapView](#endstone._internal.endstone_python.MapView)</code>) – Get the map this canvas is attached to.

```python
draw_image(x:int, y:int, image:ndarray[uint8]) -> None
```

Draw an image to the map. The image will be clipped if necessary.

#### get_base_pixel_color [#endstone.map.MapCanvas.get_base_pixel_color]

```python

```

Get a pixel from the layers below this canvas.

#### get_pixel_color [#endstone.map.MapCanvas.get_pixel_color]

```python
get_pixel_color(x:int, y:int) -> tuple[int, ...] | None
```

Get a pixel from the canvas.

If no color is set at the given position for this canvas, then None is returned and the color returned by get_base_pixel_color() is shown on the map

#### map_view [#endstone.map.MapCanvas.map_view]

```python

```

#### set_pixel_color [#endstone.map.MapCanvas.set_pixel_color]

```python

```

Draw a pixel to the canvas.

If None is used as color, then the color returned by get_base_pixel_color() is shown on the map.
