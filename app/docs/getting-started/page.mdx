# Overview

**Endstone** is a modern server software for **Minecraft: Bedrock Edition**.
It adds a clean, Paper-style **plugin API** (Python & C++) on top of the [Bedrock Dedicated Server (BDS)](https://www.minecraft.net/en-us/download/server/bedrock),
letting you extend gameplay effortlessly, from high-level features (commands, events, permissions) to advanced capabilities
like packet interception.

Endstone aims for **vanilla parity** (world generation, mob AI, behaviour packs) with a focus on performance, stability,
and an ergonomic developer experience.

## System Requirements

- **CPU:** Intel® Core™ i3-3210 or AMD A8-7600 APU (or equivalent)
- **Memory:** 4 GB RAM (minimum; more recommended for plugins/players)
- **Storage:** 180 MB to 1 GB free (worlds will grow over time)
- **Network:** Broadband internet connection

<Admonition type="note">
  These baseline requirements mirror the official [Bedrock Dedicated
  Server](https://www.minecraft.net/en-us/download/server/bedrock) page.<br/>
  Actual needs grow with player count, world size, and plugins.
</Admonition>
